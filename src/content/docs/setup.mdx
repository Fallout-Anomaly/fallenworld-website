# Downloading Fallen World

## What is Wabbajack?

[Wabbajack](https://www.wabbajack.org/) is an automated modlist installer that allows you to download and install complex setups like Fallen World with minimal manual effort. Instead of spending days following a guide and resolving conflicts yourself, Wabbajack handles the downloading, patching, and configuration of over 600 mods automatically.

It ensures that everyone has the exact same stable environment, making it the most reliable way to experience modern Fallout 4.

<div style={{marginBottom: '2rem'}}>
<TerminalCard title="Step 1 - Log into Nexus">

1. Download the latest Wabbajack release from [the official site](https://www.wabbajack.org/).
2. Open Wabbajack, click **Settings**, and then click the **Nexus Mods** button to sign in.
3. This login is required so Wabbajack can automatically download the list files.

<div style={{marginBottom: '2rem'}}>
<TerminalCard title="Step 2 - Browse Lists and Install Fallen World">

1. Click **Browse Lists** in Wabbajack.
2. Search for **Fallen World** in the browser.
3. Click on **Fallen World**.
4. Click **Download** and **Install** to download the list. See Step 3 for setting install locations.

</TerminalCard>

3. Make sure you have enough free space before continuing.

## Managing storage space

Once the installation is complete, you are free to delete the **downloads folder** or move it to a different drive to save space. 

However, please be aware that if you delete these files, you will have to **redownload every mod** when a new update for the modlist is released. If you have the extra space, we highly suggest keeping the downloads folder intact to make future updates faster.

## After installation

Congratulations! Once the Wabbajack process is complete, navigate to your installation folder, then:

- **Windows users:** Open the **FALLEN WORLD WINDOWS LAUNCHER** folder and run **Launcher.exe**
- **Linux users:** Open the **FALLEN WORLD LINUX LAUNCHER** folder and run **Launcher.exe**

The launcher will guide you through the final setup steps and prepare your game for play. For more details on the first launch, see the **Launching** section.
