# Launching Fallen World

Once the installation is complete, follow the steps below to start your journey.

---

<TerminalCard title="Step 1 - Launch with the Fallen World Launcher">

To begin your experience, you must launch the game through the **Fallen World Launcher**.

1.  Navigate to your installation folder.
2.  Open the platform-specific launcher folder:
    - **Windows:** `FALLEN WORLD WINDOWS LAUNCHER`
    - **Linux:** `FALLEN WORLD LINUX LAUNCHER`
3.  Run **Launcher.exe**.
4.  The launcher will handle any remaining setup and allow you to start the game.

:::important MCM Preset
Always remember to apply the **MCM preset** when prompted or once you reach the main menu. This ensures all mods are correctly configured for the intended experience.
:::

:::warning Use the Launcher
Do not open Mod Organizer 2. Always launch the game using the `Launcher.exe` found in the platform-specific launcher folder (`FALLEN WORLD WINDOWS LAUNCHER` or `FALLEN WORLD LINUX LAUNCHER`) to ensure all systems are correctly initialized.
:::

</TerminalCard>
